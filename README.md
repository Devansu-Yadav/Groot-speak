## 🛠️ Installation Steps

1. Clone the repository

```Bash
git clone https://github.com/Devansu-Yadav/Groot-speak.git
```

2. Change the working directory

```Bash
cd Groot-speak/
```

3. Run the app locally using the <b>Go Live Extension</b> on <b>VS Code</b>
